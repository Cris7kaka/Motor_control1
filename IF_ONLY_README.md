# IF模式专用FOC控制器

## 项目概述
本项目是基于STM32F446的磁场定向控制（FOC）电机驱动器，经过优化后专门用于IF模式（电流-频率）控制。该模式通过控制电流幅值和频率来驱动电机，无需位置传感器，适用于特定应用场景。

## 特点
- 仅保留IF模式（电流-频率开环控制）
- 优化后的代码体积更小
- 专注于IF模式的性能优化
- 保留了核心FOC算法组件（SVPWM、Clarke/Park变换等）

## 控制原理
IF模式通过以下方式工作：
1. 控制电流幅值和频率
2. 保持电流与频率的固定关系
3. 使用开环角度控制
4. 通过电流反馈形成电流闭环，实现精确的电流控制

## 主要功能
- IF模式控制算法
- SVPWM调制（带零序注入）
- Clarke/Park变换
- 电流PI控制器（无斜率限制）
- 低通滤波器

## 使用方法
1. 初始化控制器：`FOC_Init(0)`
2. 设置IF参数：`FOC_SetIFModeParams(0, current_amplitude, frequency)`
3. 在主循环中调用：`FOC_Loop(controller, dt)`

## 注意事项
- 仅支持IF模式，其他控制模式已被移除
- 控制器默认启用，无需额外使能操作
- 保留了VF模式参数设置函数以维持API兼容性
- 重点关注电流控制精度，而非速度控制
- PI控制器已移除斜率限制功能
